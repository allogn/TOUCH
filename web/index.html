In this experiment we are looking for spatial objects from given datasets that are closer to each other than the parameter epsilon. The program will call spatial join several times for different values of epsilon and size of sample from datasets. Each dataset contains 10000 objects and the range of number of objects defines the number of objects from these 10000 that are used for the experiment and joined with each other. As a result, the program will generate two graphics: the first one will show the performance of developed algorithms, another - the result (number of objects that are close enough). The performance is measured as a time for spatial join depending on the number of spatial objects. The result is the number of close enough objects depending on epsilon. As input parameters you should mention the range for epsilon values and number of objects. Optimal values are proposed by default.
